# 宜搭应用迁移工具 - 一键迁移竞品应用到宜搭

> 创建时间：2026-03-12

## 需求概述

开发一套应用迁移工具，支持从主流低代码平台（氦气云、简道云、明道云、轻流、维格表等）一键迁移应用到宜搭。包含应用结构迁移、表单字段映射、流程迁移、数据迁移等功能。

## 背景

企业使用低代码平台时，常面临以下痛点：
- **平台锁定**：数据和应用难以迁移，切换成本高
- **功能差异**：不同平台能力不同，需要适配
- **数据孤岛**：历史数据难以迁移

宜搭作为钉钉生态的低代码平台，承载了大量企业应用。开发迁移工具可以帮助企业快速从竞品迁移到宜搭，降低切换成本。

## 支持的竞品

| 平台 | 优先级 | 说明 |
|------|--------|------|
| 氦气云 | P0 | 钉钉生态内竞品，迁移需求最强烈 |
| 简道云 | P0 | 市场占有率高的低代码平台 |
| 明道云 | P1 | 企业级应用为主 |
| 轻流 | P1 | 流程类应用为主 |
| 维格表 | P2 | 表格型应用为主 |
| 飞书多维表格 | P2 | 飞书生态 |

## 技术方案

### 架构设计

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  源平台     │────▶│  迁移引擎   │────▶│   宜搭     │
│  (API/文件) │     │  (处理/映射)│     │  (创建/写入)│
└─────────────┘     └─────────────┘     └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │  迁移报告   │
                    └─────────────┘
```

### 模块设计

```
/migrator
  /core
    index.ts              # 主入口
    migrator.ts           # 迁移核心逻辑
    mapper.ts             # 字段映射器
    validator.ts          # 数据校验
  /platforms
    /hydrogen             # 氦气云
      api.ts              # API 调用
      mapper.ts           # 字段映射规则
      parser.ts           # 数据解析
    /jiandaoyun           # 简道云
    /mingdaoyun           # 明道云
    /qingflow             # 轻流
    /vika                 # 维格表
  /yida
    client.ts             # 宜搭 API 封装
    converter.ts          # 转换为宜搭格式
```

## 交付物

1. **迁移工具 Web 应用** - 交互式迁移界面
2. **迁移核心库** - 可独立使用的迁移引擎
3. **各平台适配器** - 支持的竞品适配模块
4. **使用文档** - 迁移工具使用指南

## 备注

- 复杂业务逻辑（如自定义公式、脚本）可能无法完全迁移
- 建议迁移后进行人工验证
- 部分平台需要企业管理员授权才能导出数据
