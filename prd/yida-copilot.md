# 宜搭设计器 AI 助手 - Code Copilot

> 创建时间：2026-03-12

## 需求概述

在宜搭自定义页面设计器的代码模块中，集成 AI 助手（Code Copilot）。开发者可以在编写代码时通过 AI 辅助完成代码生成、解释、调试、优化等工作。

## 背景

宜搭自定义页面支持通过 JavaScript 代码实现复杂业务逻辑，但：
- **编码门槛高**：需要熟悉宜搭 JS API
- **调试困难**：缺乏智能提示和错误检查
- **效率低下**：重复代码需要手动编写

类似 GitHub Copilot 的能力可以大幅提升开发效率，让 AI 帮开发者写代码、补全代码、解释代码。

## 技术方案

### 集成方式

由于宜搭设计器是宜搭平台的一部分，无法直接修改。我们通过以下方式实现：

1. **浏览器插件方式** - 开发浏览器插件，在宜搭页面注入 AI 助手
2. **独立工具方式** - 提供独立的 AI 编码工具，可从宜搭设计器打开
3. **宜搭自定义页面方式** - 在宜搭中创建 AI 助手应用，提供代码生成服务

**推荐方案：方案 2（独立工具 + 浏览器插件）**

### 模块设计

```
/code-copilot
  /src
    /components
      CodeEditor.tsx       # 代码编辑器增强
      ChatPanel.tsx       # 对话面板
      SuggestionBox.tsx   # 建议弹窗
      Settings.tsx        # 设置面板
    /services
      ai.ts               # AI API 调用
      completer.ts        # 代码补全
      mapper.ts           # 宜达 API 映射
    /hooks
      useAI.ts            # AI 能力 Hook
      useCompletions.ts   # 补全 Hook
    /utils
      api-docs.ts         # 宜达 API 文档
      prompt.ts           # 提示词模板
  /extension              # 浏览器插件
    manifest.json
    content.ts            # 注入脚本
    popup.ts              # 插件弹窗
```

### AI 提示词设计

```typescript
// 代码补全提示词

当前代码上下文：

\`\`\`
```

## 交付物

1. **Code Copilot 浏览器插件** - 安装后在宜搭设计器中使用
2. **AI 对话 Web 应用** - 独立使用的 AI 编码助手
3. **VS Code 插件（可选）** - 本地开发时使用
4. **使用文档** - 快速开始指南

## 备注

- AI 模型推荐使用 DeepSeek（性价比高）或 OpenAI
- 需要用户提供 API Key（可选配置自有模型）
- 浏览器插件需要适配宜搭设计器的 DOM 结构
