# 宜搭 VOC/工单提交规范化工具

> 创建时间：2026-03-12

## 需求概述

开发宜搭 VOC（Voice of Customer）和工单提交的规范化工具，帮助用户按照标准格式提交问题，提高问题解决效率。

## 背景

提交宜搭工单/VOC 时常见问题：
- **信息不完整**：缺少关键环境信息、复现步骤
- **描述不清晰**：难以理解问题所在
- **格式混乱**：不同人提交格式不一
- **缺少附件**：无法快速定位问题

类似 GitHub Issue 模板，规范化工单提交。

## 功能清单

### 1. 提交模板

| 功能 | 说明 | 优先级 |
|------|------|--------|
| Bug 模板 | 标准 Bug 提交格式 | P0 |
| 功能建议模板 | 标准功能建议格式 | P0 |
| 性能问题模板 | 性能问题专用格式 | P1 |
| 数据问题模板 | 数据异常专用格式 | P1 |

### 2. 自动收集

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 环境信息 | 自动收集宜搭版本、浏览器信息 | P0 |
| 操作日志 | 自动收集相关操作日志 | P0 |
| 截图上传 | 支持截图/录屏上传 | P0 |
| 数据导出 | 导出相关数据便于排查 | P1 |

### 3. 智能分析

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 问题分类 | 自动识别问题类型 | P1 |
| 重复检测 | 检测是否已有类似问题 | P1 |
| AI 优化 | AI 优化问题描述 | P2 |

## 技术方案

### 自动收集信息

```javascript
// 自动收集环境信息
function collectEnvironmentInfo() {
  return {
    yidaVersion: getYidaVersion(),        // 宜搭版本
    browser: getBrowserInfo(),            // 浏览器信息
    os: getOSInfo(),                     // 操作系统
    screenSize: getScreenSize(),          // 屏幕分辨率
    networkType: getNetworkType(),         // 网络类型
    userAgent: navigator.userAgent,
    appType: getCurrentAppType(),         // 当前应用
    pageUrl: window.location.href,        // 当前页面
    timestamp: new Date().toISOString(),
  };
}
```
