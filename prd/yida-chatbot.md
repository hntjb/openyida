# 宜搭 AI 对话界面组件（类 Chatbot）

> 创建时间：2026-03-12

## 需求概述

将 Vercel Chatbot (https://github.com/vercel/chatbot) 的核心交互体验移植到宜搭自定义页面，提供完整的 AI 对话界面组件，支持多模型接入、流式输出、对话管理等功能。

## 背景

Vercel Chatbot 是一个功能完整的 AI 对话应用模板，核心特性：
- Next.js App Router + React Server Components
- AI SDK 统一 API（支持 OpenAI、Anthropic、Google 等）
- shadcn/ui 风格组件 + Tailwind CSS
- 对话历史持久化
- 文件上传/图片展示
- 代码高亮渲染

宜搭自定义页面需要类似能力，用于构建企业 AI 助手、智能客服、知识库问答等场景。

## 技术方案

### 依赖库

- **AI SDK**：@ai-sdk/react（流式输出、状态管理）
- **Markdown**：react-markdown + remark-gfm
- **代码高亮**：react-syntax-highlighter
- **样式**：Tailwind CSS（兼容 shadcn/ui）

### 核心组件结构

```
/components
  /chat
    ChatWindow.tsx        # 对话主窗口
    MessageList.tsx       # 消息列表
    MessageBubble.tsx     # 消息气泡
    InputBox.tsx          # 输入框
    CodeBlock.tsx         # 代码块
    LoadingDots.tsx      # 加载动画
  /sidebar
    Sidebar.tsx           # 侧边栏
    ConversationList.tsx  # 对话列表
    NewChatButton.tsx    # 新建对话
  /settings
    SettingsPanel.tsx     # 设置面板
```

### 流式输出实现

```tsx
import { useChat } from '@ai-sdk/react';

function ChatComponent() {
  const { messages, input, handleInputChange, handleSubmit } = useChat({
    api: '/api/chat',
    onFinish: (message) => {
      // 保存到历史记录
    }
  });

  return (
    <div>
      <MessageList messages={messages} />
      <form onSubmit={handleSubmit}>
        <input 
          value={input}
          onChange={handleInputChange}
          placeholder="请输入消息..."
        />
        <button type="submit">发送</button>
      </form>
    </div>
  );
}
```

### API 代理（解决跨域）

```tsx
// 自定义页面中调用外部 AI API
async function sendMessage(messages) {
  // 通过宜搭 FaaS 或自建服务代理请求
  const response = await fetch('/api/ai-proxy', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ messages, model: 'gpt-4' })
  });
  
  return response;
}
```
