# 宜搭智能打印系统

> 创建时间：2026-03-12

## 需求概述

开发一个宜搭智能打印解决方案，支持从宜搭表单获取数据、对数据进行加工处理、并按多种场景模板打印输出（红头文件、零售书本、小票等）。

## 背景

宜搭平台自带打印模板功能，但存在以下局限：
1. **模板样式固定** - 仅支持基础富文本编辑，难以实现复杂排版
2. **数据加工能力弱** - 无法对原始数据进行二次计算、格式化、汇总
3. **场景支持单一** - 缺乏针对红头文件、书籍、小票等特定场景的模板库
4. **批量打印效率低** - 每次最多100条，无数据分组聚合功能

本需求旨在创建一个完整的打印解决方案，覆盖企业办公、零售销售、仓储物流等多场景。

## 功能清单

### 1. 数据源管理

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 表单选择 | 选择宜搭应用和表单作为数据源 | P0 |
| 字段映射 | 将表单字段映射到打印模板变量 | P0 |
| 数据过滤 | 支持按条件筛选数据（日期范围、状态等） | P0 |
| 数据关联 | 支持多表单关联查询（主表+子表） | P1 |
| 数据聚合 | 支持对数据进行求和、计数、平均等计算 | P1 |

### 2. 数据加工引擎

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 文本格式化 | 日期格式转换、数字格式化、金额转大写 | P0 |
| 数据计算 | 表达式计算（单价×数量=金额） | P0 |
| 数据分组 | 按类别、部门、时间段分组 | P1 |
| 数据汇总 | 自动计算小计、合计、总计 | P1 |
| 条码/二维码 | 生成商品条码、二维码用于打印 | P1 |
| 图片处理 | 裁剪、缩放、水印 | P2 |

### 3. 打印模板库

#### 3.1 红头文件类

| 模板名称 | 说明 |

| 正式公文 | A4 纵向，红头标题、文号、主送、抄送、正文、落款 |

| 会议纪要 | 包含参会人员、议程、决议事项 |
| 工作报告 | 标题、摘要、正文、附件说明 |
| 证明文件 | 证照类打印，支持盖章区 |

#### 3.2 零售书本类

| 模板名称 | 说明 |
|----------|------|
| 商品标签 | 价格标签、条码、规格、产地 |
| 销售小票 | 收银小票格式，包含商品明细、金额、找零 |
| 出库单 | 仓库出库单，包含商品列表、数量、签名区 |
| 盘点清单 | 库存盘点表，支持数量录入区域 |
| 订货单 | 采购订单，包含商品、单价、数量、金额 |

#### 3.3 票据凭证类

| 模板名称 | 说明 |
|----------|------|
| 收据 | 收款收据，支持二联/三联 |
| 发票模板 | 模拟发票格式（不含税控） |
| 快递单 | 物流面单模板 |
| 合格证 | 产品合格证，支持Logo、规格、日期 |

### 4. 打印控制

| 功能 | 说明 | 优先级 |
|------|------|--------|

| 打印方向 | 纵向/横向 | P0 |
| 预览功能 | 打印前预览效果 | P0 |

## 技术方案

### 架构设计

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   宜搭表单      │────▶│  数据加工引擎   │────▶│  打印渲染器    │
│  (数据源)       │     │  (处理/计算)    │     │  (模板引擎)    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                                                        │
                                                        ▼
                                               ┌─────────────────┐
                                               │  浏览器打印/    │
                                               │  PDF 导出       │
                                               └─────────────────┘
```

### 技术选型

- **模板引擎**：Handlebars / EJS
- **PDF 生成**：jsPDF + html2canvas
- **条码生成**：JsBarcode + qrcode
- **打印控制**：window.print() + CSS @media print

### 模板变量规范

```javascript
// 模板中使用双花括号作为变量标记
{{field_name}}              // 普通字段
{{formatDate date "YYYY-MM-DD"}}  // 日期格式化
{{currency amount}}        // 金额格式化
{{upperCase text}}         // 文本转大写
{{#each items}}...{{/each}} // 循环遍历
{{sum items "price"}}      // 求和计算
```
