# 宜搭二维码批量打印系统

> 创建时间：2026-03-12

## 需求概述

开发宜搭二维码批量打印解决方案，支持从表单数据批量生成二维码、多种排版模式、自定义Logo/品牌元素，参考草料二维码、BarTender等成熟产品实现。

## 背景

企业场景中大量需要二维码标签：资产盘点、产品追溯、设备巡检、物料管理等。宜搭现有能力无法满足：
- 缺乏批量二维码生成能力
- 排版模式单一
- 无法自定义品牌元素（Logo、颜色、样式）

参考产品：草料二维码、BarTender、智高标签打印软件

## 功能清单

### 1. 二维码生成

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 文本转二维码 | 将文本/数字直接转为二维码 | P0 |
| 链接转二维码 | URL 转为可扫码跳转的二维码 | P0 |
| 表单字段转二维码 | 将宜搭表单字段值转为二维码 | P0 |
| 批量生成 | 从数据列表批量生成二维码 | P0 |
| 二维码美化 | 颜色、渐变、形状、logo 定制 | P1 |
| 条码生成 | 支持 Code128、EAN13 等一维码 | P1 |

### 2. 排版模式

| 模式 | 说明 | 示例场景 |
|------|------|----------|
| 单排横向 | 一行一个，从左到右 | 资产标签 |
| 单排纵向 | 一列一个，从上到下 | 图书标签 |
| 多列多行 | 行列矩阵排列 | A4 纸多标签 |
| 重复平铺 | 同一标签重复打印 N 份 | 库存标签 |
| 流水排列 | 横向蛇形排列 | 生产批次标签 |
| 自定义网格 | 指定行列数和间距 | 特殊标签纸 |

### 3. 标签设计

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 添加文字 | 静态文字 + 动态变量 | P0 |
| 添加图片/Logo | 上传企业 Logo | P0 |
| 添加边框 | 矩形、圆角、无边框 | P1 |
| 添加背景色 | 自定义背景颜色 | P1 |
| 尺寸设置 | 标签宽度、高度、边距 | P0 |
| 条码组合 | 二维码 + 一维码组合 | P2 |

### 4. 数据源

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 手动输入 | 逐条输入文本内容 | P0 |
| 文本批量导入 | 多行文本一次性导入 | P0 |
| Excel 导入 | 导入 Excel 表格数据 | P0 |
| 宜搭表单 | 直接读取宜搭表单数据 | P1 |
| 流水号生成 | 自动生成递增序号 | P1 |

### 5. 打印输出

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 打印预览 | 打印前预览效果 | P0 |
| 纸张尺寸 | A4、A5、自定义 | P0 |
| 打印份数 | 每种标签打印份数 | P0 |
| PDF 导出 | 导出 PDF 文件 | P1 |
| 图片导出 | 导出 PNG/JPG 图片 | P1 |

## 技术方案

### 依赖库

- **二维码生成**：qrcode
- **一维码生成**：JsBarcode
- **PDF 生成**：jsPDF + jspdf-autotable
- **Excel 解析**：xlsx

### 模板变量

```javascript
// 支持的变量
{{serialNumber}}     // 流水号（自动递增）
{{rowNo}}           // 行号（从1开始）
{{colNo}}           // 列号
{{field_name}}      // Excel/表单字段值
{{date}}            // 当前日期
{{time}}            // 当前时间
```

## 备注

- 建议使用 Chrome 浏览器和激光打印机获得最佳效果
- 标签纸规格需与打印机匹配
- 大量数据建议分批处理（单次建议 < 1000 个）
